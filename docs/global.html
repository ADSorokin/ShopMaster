AddToCartAnimation(props)

Поведение:

  • Отображается как всплывающее уведомление в правом верхнем углу
  • Появляется при добавлении товара в корзину
  • Имеет анимацию скрытия (через CSS-класс animate-fade-out)
  • Автоматически исчезает через 1.5 секунды (управляется извне)
  • Поддерживает мультиязычный текст

Примечание: для работы анимации требуется CSS-анимация animate-fade-out.
