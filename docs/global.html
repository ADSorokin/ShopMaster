Страница поддерживает:

OrdersPage(props)

Поведение:

  • Если пользователь не авторизован — показывает сообщение и кнопку входа.
  • Если нет заказов — отображает пустое состояние с призывом к покупкам.
  • Иначе — рендерит список заказов с деталями: статус, доставка, оплата, товары.

Поддерживает мультиязычность через объект translations.

ProductCard(props) → {JSX.Element}

Компонент карточки товара

Отображает карточку с информацией о товаре, включая изображение, название, описание, рейтинг, цену и кнопки действий (добавить в корзину, избранное, сравнение).

При клике на карточку (не на кнопки) происходит переход к странице товара.
