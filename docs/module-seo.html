Module: seo

Утилита для генерации SEO-данных (title, description, keywords) в зависимости от страницы.

Source:
  • utils/seo.js, line 1
Examples
const seo = getSeoData('electronics');
document.title = seo.title;
const seo = getSeoData('product', product);
updateMetaTags(seo);

Members

(static, constant) getSeoData

Особенности:

  • Для страниц категорий возвращает предопределённые SEO-тексты
  • Для страницы товара генерирует динамические мета-данные
  • Автоматически использует русские названия (для SEO)
  • При неизвестной странице возвращает данные для главной
Properties:
Name Type Description
title string

Заголовок страницы (до 60 символов)
description string

Описание для поисковых систем (до 160 символов)
keywords string

Ключевые слова через запятую
Source:
  • utils/seo.js, line 46
Examples
getSeoData('home');
// → { title: "ShopMaster - Лучший интернет-магазин...", ... }
getSeoData('product', {
  name: { ru: 'Смартфон X1' },
  description: { ru: 'Флагманский смартфон...' },
  price: 49990,
  brand: 'TechPro',
  specifications: { ru: { partNumber: 'SMX1-2024' } }
});
// → { title: "Смартфон X1 - купить в интернет-магазине ShopMaster", ... }
